# Automated Data Storytelling for E-commerce

## Project Goal
Automatically calculate weekly KPIs for an online shop and generate human-readable insights.

## Domain
E-commerce

## KPIs
- Total Sales
- Number of Orders
- Average Order Value

## Project Structure
- `data/` - raw and processed datasets
- `notebooks/` - Jupyter notebooks for exploration
- `src/` - reusable Python scripts
- `reports/` - plots and generated insights

## Getting Started
1. Clone the repo
2. Install dependencies: `pip install -r requirements.txt`
3. Open notebooks in Jupyter Lab or VS Code
